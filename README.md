# iOS Dining Companion App

A native iOS app that learns your taste profile and helps you decide 
what to order — anywhere food is served.

## What it does

- Scans menus using on-device OCR
- Matches dishes against a personal taste profile built over time
- Surfaces personalized recommendations with context on why they fit
- Logs meals and refines recommendations based on feedback

## Stack

- SwiftUI / SwiftData
- Apple Vision (on-device OCR)
- Claude API (recommendation engine)
- Supabase (backend / Edge Functions)

## Status

In TestFlight with a private beta group. Pre-App Store.
